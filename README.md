# Tad

**Tad** is a framework for terminal based user interface development, unifying TUIs and giving users the options to customize the apps they use on a daily basis.

## Development

To install dependencies:

```bash
bun install
```

To run:

```bash
bun .
```

This project was created using `bun init` in bun v1.3.0. [Bun](https://bun.com) is a fast all-in-one JavaScript runtime.
